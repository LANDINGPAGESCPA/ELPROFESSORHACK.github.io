  • How to fix Untrusted Enterprise Developer?

    In order to use any application install from Ignition, you will need to trust the enterprise certificate we are currently using.

    In order to trust these certificates you need to go to Settings -> General -> Profiles or Profiles and Device Management. Under the Enterprise App title, you will see a link which say the name on the Untrusted Enterprise Developer. Tap this link and tap on Trust. You will need to confirm you want to trust the profile, after doing that apps should now launch.

  • Why do I get the "Unable to Download App" error?

    Usually this error may happen when Apple has revoked our certificate.

    If Apple has revoked our certificate you will have to wait a few hours for us to fix this.

    Another reason may be that you already have an app install with the same Bundle ID.

    To fix this you need to remove any versions of the app you are trying to install that you may currently have on your device.

  • I don't get a prompt to install the app.

    Sometimes it may take a few seconds for the prompt to show up due to server load or internet speeds.

    So, just be patient. If you have been waiting for more than a minute try restarting Ignition.

    If that doesn't work, try a new internet connection and try again.

    Finally, if all else fails, send us a Direct Message on Twitter @ignition and we will try help you solve the problem.
